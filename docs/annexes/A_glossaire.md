# Annexe A : Glossaire

## A.1 Termes généraux

| Terme | Définition |
|-------|------------|
| **ANStat** | Agence Nationale de la Statistique (ancien INS) - Institut statistique national de Côte d'Ivoire |
| **Big Data** | Ensemble de données massives caractérisées par les 3V : Volume, Vélocité, Variété |
| **CDR** | Call Detail Records - Enregistrements détaillés des appels téléphoniques |
| **Dashboard** | Tableau de bord interactif permettant la visualisation des données |
| **Data Science** | Science des données - Discipline combinant statistique, informatique et expertise métier |
| **GADM** | Global Administrative Areas Database - Base de données des limites administratives mondiales |
| **GPS** | Global Positioning System - Système de géolocalisation par satellite |
| **H3** | Système d'indexation géospatiale hexagonale développé par Uber |
| **INS** | Institut National de la Statistique (ancien nom de l'ANStat) |
| **IoT** | Internet of Things - Internet des objets |
| **ML** | Machine Learning - Apprentissage automatique |
| **ODD** | Objectifs de Développement Durable des Nations Unies |
| **RGPH** | Recensement Général de la Population et de l'Habitat |
| **SNDS** | Système National de Développement de la Statistique |
| **UN** | United Nations - Nations Unies |

## A.2 Termes statistiques

| Terme | Définition |
|-------|------------|
| **ACP / PCA** | Analyse en Composantes Principales - Méthode de réduction dimensionnelle |
| **Alkire-Foster** | Méthode de calcul de l'Indice de Pauvreté Multidimensionnelle |
| **Centile** | Division d'une distribution en 100 parties égales |
| **Coefficient de Gini** | Mesure des inégalités (0 = égalité parfaite, 1 = inégalité maximale) |
| **Corrélation** | Mesure de la relation linéaire entre deux variables |
| **Décile** | Division d'une distribution en 10 parties égales |
| **Écart-type** | Mesure de dispersion autour de la moyenne |
| **ENV** | Enquête sur le Niveau de Vie des ménages |
| **Incidence (H)** | Proportion de la population identifiée comme pauvre |
| **Intensité (A)** | Moyenne des privations parmi les pauvres |
| **IPM / MPI** | Indice de Pauvreté Multidimensionnelle |
| **Médiane** | Valeur centrale d'une distribution ordonnée |
| **Moyenne** | Somme des valeurs divisée par leur nombre |
| **OPHI** | Oxford Poverty and Human Development Initiative |
| **Quintile** | Division d'une distribution en 5 parties égales |
| **R²** | Coefficient de détermination - Part de variance expliquée |
| **Régression** | Méthode d'estimation de la relation entre variables |
| **Variance** | Mesure de la dispersion des données |
| **Wealth Index** | Indice de richesse - Proxy du niveau socio-économique |

## A.3 Termes de mobilité

| Terme | Définition |
|-------|------------|
| **agriculture_saisonniere** | Migration temporaire liée aux cycles agricoles |
| **Corridor migratoire** | Axe géographique concentrant des flux de migration |
| **Domicile** | Lieu de résidence habituel détecté par analyse des positions nocturnes |
| **Heure de pointe** | Période de concentration maximale des déplacements |
| **Matrice O-D** | Matrice Origine-Destination des flux de déplacement |
| **Migration** | Changement de résidence habituelle (distance > 50 km, durée > 30 jours) |
| **migration_circulaire** | Déplacements répétés entre deux lieux |
| **migration_etudes** | Migration motivée par la poursuite d'études |
| **migration_travail** | Migration motivée par l'emploi |
| **Mobilité quotidienne** | Ensemble des déplacements journaliers |
| **Mode de transport** | Moyen utilisé pour se déplacer (marche, moto, taxi, bus, voiture) |
| **Motif de déplacement** | Raison du trajet (travail, courses, loisirs, etc.) |
| **Rayon de mobilité** | Distance maximale parcourue depuis le domicile |
| **relocalisation_permanente** | Migration définitive vers un nouveau lieu |
| **Trajet** | Déplacement entre deux points |

## A.4 Termes techniques

| Terme | Définition |
|-------|------------|
| **API** | Application Programming Interface - Interface de programmation |
| **CSV** | Comma-Separated Values - Format de fichier texte |
| **DataFrame** | Structure de données tabulaire (lignes et colonnes) |
| **FastAPI** | Framework Python pour créer des API web |
| **GeoJSON** | Format de données géographiques basé sur JSON |
| **JSON** | JavaScript Object Notation - Format d'échange de données |
| **k-anonymat** | Technique de protection de la vie privée |
| **Parquet** | Format de stockage de données en colonnes |
| **Pipeline** | Chaîne de traitement automatisée |
| **Python** | Langage de programmation utilisé pour ce projet |
| **REST** | Representational State Transfer - Architecture d'API web |
| **Streamlit** | Framework Python pour créer des applications web de données |
| **XLSX** | Format de fichier Microsoft Excel |
